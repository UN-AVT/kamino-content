Is LatLong

Check if the string is a valid latitude-longitude coordinate. The latitude of the location ranges from -90.0 to 90.0. The longitude of the location ranges from -180.0 to 180.0.

Let’s check if “latitude” column fit the requirement.

df$latitude<90.0 & df$latitude>-90.0
##  [1] TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE
## [16] TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE

As you can see, the column fits the latitude requirement.

Now, let’s check if “longitude” column fit the requirement.

df$longitude < 180.0 & df$longitude > -180.0
##  [1] TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE
## [16] TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE
