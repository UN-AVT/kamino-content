Is Hex Color

Hexadecimal Color Code starts with a # followed by six or three letters or digits.For example: #fff, #FFF, #000, #111111, #F000F0, #00FF00, #0000FF, #FFFFFF are all valid Hexadecimal color codes.

To validate, we can use the function grepl()

a <- df$Hex_code

df_1 <- grepl("^#([A-Fa-f0-9]{6}|[A-Fa-f0-9]{3})$", a, perl = T)
df_1
##  [1] TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE
## [16] TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE TRUE

As you can see, every cell in column “Hex_code” are valid Hexadecimal Codes.
