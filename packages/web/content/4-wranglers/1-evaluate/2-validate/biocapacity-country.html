Wrangle

ISO_Alpha2

ISO 3166-1 alpha-2 codes are standardized two-letter country codes. We can validate ISO2 codes by using library(countrycode). Let’s enter “iso2c” as origin first (meaning we guess it is ISO-A2). For each cell, it will return the country name if the code is recognized. To do this, we use the function countrycode()

results <- countrycode(df$isoa2, origin = 'iso2c', destination = 'country.name')
df2 <- data.frame(df$isoa2, results)
df2

Our package successfully converted all our iso2c codes from the ingested data into Country Names. This means that our column follows the iso2c code.
