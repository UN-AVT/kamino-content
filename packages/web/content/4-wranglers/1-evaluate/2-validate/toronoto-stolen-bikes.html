Is is_ISO8601

ISO 8601 is an international standard covering the worldwide exchange and communication of date- and time-related data in the format yyyy-mm-dd or yyyy-mm-dd hh:mm:ss. To validate this we input the column into a try-catch with the function as.Date()

d <- try(as.Date(df$DATE, format="%Y-%m-%d"))
if("try-error" %in% class(d) || is.na(d)) {
  print("The list is NOT in an ISO8601 format")
}else print("The list is in an ISO8601 format")
## [1] "The list is in an ISO8601 format"
