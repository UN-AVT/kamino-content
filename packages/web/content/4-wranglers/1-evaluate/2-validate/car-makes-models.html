Wrangle

Is In List

We will check if a list contains a particular element or a multitude of elements. Here, we ingested a list of automobiles and their features. We’ll see if a particular car is in that list. We can use the operator %in%

"bmw x5" %in% df$CarName
## [1] TRUE

We can see that the string ‘bmw x5’ is in our ingested dataset. We can also create a list of cars to check if they are in the dataset.

a <- c("nissan sentra", "honda civic", 'toyota corolla', 'hyundai elantra')
a
## [1] "nissan sentra"   "honda civic"     "toyota corolla"  "hyundai elantra"
a %in% df$CarName
## [1] FALSE  TRUE  TRUE FALSE
