The United States is among the most expensive for mobile data. According to a study by UK based research firm businessfibre, the US follows Switzerland, South Korea and Norway among the nations with the most expensive broadband and mobile internet charges.

Ingest the data

Rank, Country, and Avg_Price_of_1GB_USD.

url_root <- "https://raw.githubusercontent.com/UN-AVT/kamino-source/main/sources/0-shared/data/"
url_file <- "cost-of-mobile-data/cost-of-1gb-of-data.csv"
url <- paste0(url_root, url_file)

df <- read.csv(url, header = TRUE, stringsAsFactors = FALSE)
df

Wrangle

We can round numbers to specified decimals, let’s round Avg_Price_of_1GB_USD to 1 decimal using round() function.

df1 <- df
df1$Avg_Price_of_1GB_USD <- round(df1$Avg_Price_of_1GB_USD,digits = 1)
df1

We can also use format() function in combination with the round() function. The output of the format function is a character string.For performing calculations, we need to convert it back to the numeric class.

df2 <- df 

df2$Avg_Price_of_1GB_USD <- format(round(df2$Avg_Price_of_1GB_USD,1),nsmall = 1)
df2

The sprintf() R function also provides the possibility to control decimal places of a certain number or a numeric vector.

df3 <- df

df3$Avg_Price_of_1GB_USD <- sprintf(df3$Avg_Price_of_1GB_USD, fmt='%#.1f')
df3
