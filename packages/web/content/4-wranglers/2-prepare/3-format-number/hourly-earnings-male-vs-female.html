Ingest the data

Entity, Year, population, avg_hourly_wage_men, and avg_hourly_wage_women

url_root <- "https://raw.githubusercontent.com/UN-AVT/kamino-source/main/sources/0-shared/data/"
url_file <- "hourly-earnings/hourly-earnings-male-vs-female.csv"
url <- paste0(url_root, url_file)

df <- read.csv(url, header = TRUE, stringsAsFactors = FALSE)
df

Wrangle

First of all, let’s remove all “NA” values, do you remember how we did in Clean?

df_1 <- na.omit(df)

rownames(df_1) <- NULL

Let’s use following steps to calculate the percentage of increase/decrease for each year.

  • Calculate the difference of average hourly wage between each year for men and women;
  • Use the difference to divide the previous average hourly wage of that year then times 100;
  • Round the result and add “%” sign in the end.
df_2<-df_1%>%
  group_by(Entity)%>%
  mutate(avg_hourly_wage_diff_men=(avg_hourly_wage_men/lag(avg_hourly_wage_men) - 1)*100)%>%
  mutate(avg_hourly_wage_diff_women=(avg_hourly_wage_women/lag(avg_hourly_wage_women) - 1)*100)%>%
  dplyr::select(Entity,Year,avg_hourly_wage_diff_men,avg_hourly_wage_diff_women)

df_2 <- na.omit(df_2)

  df_2$avg_hourly_wage_diff_men<-format(round(df_2$avg_hourly_wage_diff_men,1), nsmall=1)
  df_2$avg_hourly_wage_diff_men<-paste0(df_2$avg_hourly_wage_diff_men," %")
  
  df_2$avg_hourly_wage_diff_women<-format(round(df_2$avg_hourly_wage_diff_women,1), nsmall=1)
  df_2$avg_hourly_wage_diff_women<-paste0(df_2$avg_hourly_wage_diff_women," %")
