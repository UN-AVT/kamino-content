Scatter Contour

Story-Telling with Scatter Contour

Scatter plots’ primary uses are to observe and show relationships between two numeric variables. A contour plot is appropriate if you want to see how some value Z changes as a function of two inputs, X and Y: z = f(x, y). Scatter-Contour provides a better visualization result as the contour which commonly show altitude (like height of a geographical features), can also be used to show density, brightness, or electric potential. Inclusion of contour are particularly helpful when you need to identify combinations of X and Y that produce beneficial or required values of Z. The contour lines and bands make it easy to find combinations that yield the values you need.

Contour plots are helpful to assess the following features of the dataset:

  • Explore the relationship between three variables on a single chart
  • View combinations of X and Y that produce desirable outcome values

Types of Contour plot:

  • Rectangular Contour : Projection of 2-D plot in 2-D rectangular canvas.
  • Polar Contour : They are circular and plotted by using the polar coordinates r and theta.
  • Ternary Contour : They are triangular and show a relationship between three explanatory variables and a response variable. Most commonly, the third explanatory variable is a height value for an XYZ value in ternary space.
