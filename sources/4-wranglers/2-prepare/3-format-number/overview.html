Format Numbers

Sometimes we need to format number to different types for better understanding, more readable or visual purpose. In our example, all numbers are in double class, meaning they are double-precision floating point numbers. Ways to format numbers:

  • Scientific format: better for large numbers
  • Comma separators format: better for large numbers
  • Long to readable: better for large numbers
  • Currency format: better for financial numbers
  • Decimal format: better for consistence or visual purpose
  • Percentage format: better for showing an increasing or decreasing trend
  • Fraction format: better for comparison
